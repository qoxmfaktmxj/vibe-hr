# Vibe-HR 아이콘 가이드 (v1)

목표: 메뉴 인지성 + 도메인 일관성 유지

## Top-level
- `dashboard` → `LayoutDashboard`
- `hr` → `UsersRound`
- `tim` → `Clock`
- `payroll` → `Wallet`
- `settings` → `Settings`

## 2nd/3rd-level
- `hr.employee` → `UserRound`
- `hr.attendance` → `CalendarCheck2`
- `hr.leave` → `CalendarDays`
- `payroll.calc` → `Calculator`
- `payroll.slip` → `FileText`
- `settings.roles` → `Shield`
- `settings.permissions` → `Menu`
- `settings.menus` → `PanelLeft`

## 원칙
1. 상위 메뉴는 도메인 아이콘 1개로 고정
2. 하위 메뉴는 기능 성격이 바로 보이는 아이콘 사용
3. 동일 화면군 내 아이콘 중복 최소화
4. 새 메뉴 추가 시 본 가이드 우선 적용

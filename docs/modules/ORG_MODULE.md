# 조직(ORG) 모듈 상세 문서

## 목적
법인/조직/부서 구조를 표준화하고, 인사/근태/신청 모듈의 기준 조직정보를 제공한다.

## 현재 구현 범위
- 부서 CRUD
- 조직 트리 조회
- 코드형 조직 분류 관리(유형/업로드)

## 주요 경로
- Backend API: `backend/app/api/organization.py`
- Frontend: `frontend/src/app/org/*`

## 데이터 모델 핵심
- `org_departments`
- 조직 관련 분류 테이블(코드 기반)

## 변경 시 주의
1. 부서 ID는 HR/TIM/HRI/CPN 전체 FK로 사용됨
2. 부서 삭제 시 참조 무결성 영향 범위 큼(직원/신청/스케줄)
3. 조직명 변경은 표시 이슈(캐시/메뉴 라벨) 재검증 필요

## 테스트 체크
- 조직 트리 조회 일관성
- 부서 생성/수정/비활성
- 타 모듈 연동 화면에서 부서 필터 정상 노출

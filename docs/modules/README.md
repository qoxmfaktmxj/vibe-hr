## 문서 동기화 정책 (중요)
- 신규 화면/신규 API/신규 테이블이 추가되면 **반드시 해당 모듈 MD를 같은 커밋(또는 직후 커밋)에서 갱신한다.**
- `git pull`로 반영된 변경 중 MD 누락이 보이면, 구현 변경이 없어도 MD 보완 커밋을 추가한다.
- 최소 기록 항목: 변경 목적, 파일 경로, 데이터 영향, 테스트 포인트.

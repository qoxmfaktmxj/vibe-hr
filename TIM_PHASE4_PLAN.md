# TIM Phase 4 - 통계/리포트 구현 계획

## 목표
- 관리자용 근태 통계 대시보드 제공
- 부서별 출근율/지각율/결근율 집계
- 휴가 신청 유형별 집계 제공

## 구현 범위 (이번 반영)
1. Backend
   - `GET /api/v1/tim/reports/summary`
   - 기간 파라미터: `start_date`, `end_date` (미입력 시 최근 30일)
   - 응답: 상태별 카운트, 부서별 비율, 휴가유형 집계

2. Frontend
   - 페이지: `/tim/reports`
   - 메뉴: `근태리포트` (hr_manager/admin)
   - 카드 + 테이블 기반 대시보드

3. 비범위
   - 차트 라이브러리 연동
   - 엑셀 다중 템플릿 다운로드
   - 법정수당 계산(야간/휴일근로)

## 후속(Phase 4 확장)
- 주52시간 초과 리포트
- 연차 미사용 리포트
